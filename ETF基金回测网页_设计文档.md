# ETF 基金回测网页 — 设计文档

## 1. 项目概述

### 1.1 目标
构建一个面向个人投资者的 ETF 基金回测网页工具，用户可以选择 ETF 标的、设定投资策略，直观地看到历史收益表现和风险指标，辅助投资决策。

### 1.2 核心用户
- 个人投资者、基金定投用户
- 对指数投资感兴趣但没有编程背景的用户

### 1.3 核心价值
- **低门槛**：无需编程，网页操作即可完成回测
- **可视化**：图表展示收益曲线，直观易懂
- **灵活**：支持多种定投策略和参数配置

### 2.2 详细功能描述

#### F1: 基金选择
- **热门 ETF 列表**：预设 10-15 只主流 ETF（A股、港股、美股），点击即可选择
- **搜索/输入**：支持通过基金代码或名称搜索
- **多标的对比**：支持同时选择 2-3 只 ETF 进行对比回测

#### F2: 参数配置
- **回测时间**：日期选择器，默认最近 3 年，范围限制在数据可用范围内
- **投资策略**：
  - **一次性投资**：在开始日期投入全部资金
  - **定期定额**：经典的定投策略，每期固定金额
  - **定期不定额**：根据估值指标（PE/PB）调整每期金额，低估值多投，高估值少投
- **交易成本**：申购费率（默认 0.15%）、管理费率（默认 0.5%/年）、赎回费率

#### F3: 结果展示
- **收益概览卡片**：核心指标以卡片形式展示，含同比/对比色标识好坏
- **资产净值曲线图**：主图展示资产总值 vs 累计投入 vs 基准（可选沪深300指数）
- **定投明细表**：可排序、搜索的交易明细表，支持导出 CSV

### 4.3 核心模块

#### 回测引擎 (BacktestEngine)
```
class BacktestEngine:
    - 输入: 净值数据, 策略参数
    - 执行:
        1. 按时间顺序遍历交易日
        2. 在定投日执行买入（按策略计算金额）
        3. 记录每笔交易
        4. 每日计算持仓市值
    - 输出: 交易记录, 每日资产曲线, 汇总指标
```

##### 策略计算逻辑

**定期定额策略**（核心，最简实现）：
```
每期买入金额 = 固定值（用户设定）
每期买入份额 = 买入金额 / 当日净值
持仓份额 = sum(所有买入份额)
持仓市值 = 持仓份额 × 当前净值
累计投入 = sum(所有买入金额)
累计收益 = 持仓市值 - 累计投入
累计收益率 = 累计收益 / 累计投入 × 100%
```

**一次性投资策略**：
```
开始日买入份额 = 总投入金额 / 当日净值
持仓市值 = 持仓份额 × 当前净值
```

##### 指标计算
| 指标 | 计算方法 |
|---|---|
| 累计收益率 | (最终价值 - 总投入) / 总投入 × 100% |
| 年化收益率 | (最终价值 / 总投入) ^ (1/年数) - 1 |
| 最大回撤 | max(1 - 当前净值/区间内净值最高点) |
| 年化波动率 | 日收益率标准差 × √252 |
| 夏普比率 | (年化收益率 - 无风险利率) / 年化波动率 |

## 9. 风险与注意事项

### 9.1 数据风险
- 基金净值数据的准确性和完整性是关键依赖
- 历史数据不代表未来收益
- 需要明确展示**免责声明**

### 9.2 技术风险
- 纯前端计算大量日频数据时可能存在性能问题（10年 ≈ 2500 个交易日，单只 ETF 计算量 OK）
- 内置 CSV 数据会增加打包体积（建议 gzip，约 200KB/只 ETF）

### 9.3 合规
- 页面必须有明确的投资风险提示
- 不得承诺收益
- 标明数据来源和免责声明

## 附录 A：回测引擎核心算法伪代码

```
function runBacktest(navData, config):
    // navData: [{date, nav}, ...] 按日期升序
    // config: {startDate, endDate, strategy, amount, frequency, fee}
    
    let cashflow = []
    let dailyValues = []
    let shares = 0
    let totalInvested = 0
    
    for each day in navData:
        if day.date < config.startDate: continue
        if day.date > config.endDate: break
        
        // 判断是否为定投日
        if isInvestmentDay(day.date, config):
            fee = config.amount * config.feeRate
            actualInvest = config.amount - fee
            sharesBought = actualInvest / day.nav
            shares += sharesBought
            totalInvested += config.amount
            cashflow.push({date: day.date, type: 'buy', amount: config.amount, ...})
        
        // 记录每日持仓价值
        dailyValues.push({
            date: day.date,
            invested: totalInvested,
            value: shares * day.nav
        })
    
    // 计算最终结果
    finalValue = shares * navData.last().nav
    return {
        cashflow,
        dailyValues,
        totalInvested,
        finalValue,
        totalReturn: (finalValue - totalInvested) / totalInvested,
        // ... 其他指标
    }
```

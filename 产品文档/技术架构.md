## 回测引擎

### 单基金引擎 (`src/engine/strategies.ts`)
纯 JS 同步计算，遍历每个交易日执行：
1. 计算周期收益率 (cycleReturn)
2. 补仓检查（亏损触发追加买入）
3. 定投/买入（含智能定投调额）
4. 移动止盈检查（从最高点回落触发卖出）
5. 止损检查（亏损触发卖出）

### 组合引擎 (`src/engine/portfolio.ts`)
- 逐只基金调用 runBacktest()
- 按日期聚合 dailyValues
- 计算汇总指标

### 核心状态变量
| 变量 | 说明 |
|------|------|
| totalShares | 累计持仓份额 |
| totalInvested | 累计投入金额（只增不减） |
| cashBalance | 现金余额（卖出所得） |
| cycleBasisInvested | 当前周期起始累计投入 |
| peakCycleReturn | 周期内最高收益率（只增不减） |
| tpTriggered/slTr/dbTr | 已触发的阈值集合 |

## 数据流

### 单基金回测
```
选择基金 + 配置参数 → loadNavData → CSV → NavRecord[]
→ runBacktest(NavRecord[], config) → BacktestResult
→ 指标卡片 + 图表 + 交易明细表
```

### 组合回测
```
配置资金 + 添加基金 + 各基金策略 → runPortfolioBacktest
→ 逐只 loadNavData + runBacktest → 按日期聚合 → 组合指标
```

## 数据持久化

| 存储 | Key | 内容 |
|------|-----|------|
| localStorage | portfolio_config | 组合配置（资金 + 基金列表 + 策略） |
| localStorage | custom_etfs | 用户自定义添加的基金 |
| 模块变量 | _cachedResult | 组合回测结果（切换 Tab 不丢失） |
| CSV 文件 | public/data/prices/{code}.csv | 基金净值数据 |

## API 服务 (`api_server.py`)
- 框架：FastAPI，端口 8765
- 搜索：fund_open_fund_daily_em() 全市场场外基金（2 万+）
- 拉取：fund_open_fund_info_em() 基金净值
- 列表缓存 1 小时

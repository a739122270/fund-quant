# 数据源说明

## 来源
天天基金网（东方财富），通过 AkShare 接口获取。

### 基金净值
- 接口：`fund_open_fund_info_em()`
- 数据：场外基金单位净值，日频
- 字段：date, close（单位净值）

### 基金列表
- 接口：`fund_open_fund_daily_em()`
- 数据：全市场场外基金（约 2.4 万只）
- 搜索：通过 API `/api/funds/search?q=` 实时查询，缓存 1 小时

## 数据格式
```csv
date,open,close,high,low,volume,amount
2023-01-03,,3.9521,,,,
```
核心字段为 close（单位净值），其余留空。

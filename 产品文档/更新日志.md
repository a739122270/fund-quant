# 更新日志

## [Unreleased]

### 新增
- **组合回测页面**：支持多只基金同时回测，设定初始金额和每月可用金额
- **全市场基金搜索**：实时查询 2 万+ 场外基金，搜索即拉取带反馈
- **基金编辑删除**：编辑模式 + 红色圆圈 ➖ 删除按钮
- **组合配置持久化**：localStorage 保存组合配置，切换 Tab 不丢失
- **回测结果缓存**：模块变量缓存，切换 Tab 回来结果仍在
- **Tab 切换**：单基金回测 / 组合回测 双页面 Tab 切换
- **定投顺延**：非交易日自动顺延到下一个交易日
- **各基金收益贡献表**：总资产、收益、收益率、占比

### 变更
- 预设基金从 15 只场内 ETF 替换为 15 只场外基金
- 搜索只搜已加载基金，全量搜索移入"添加基金"功能
- 补仓触发条件从"净值回撤"改为"持仓亏损"
- 搜索和添加流程改为异步带反馈（⟳ 拉取中 / ✓ 已添加）
- 回测引擎支持多策略组合（智能定投 + 补仓 + 止盈 + 止损）
- 总资产计算统一为"持仓市值 + 现金余额"
- ConfigPanel 新增 `cardTitle` 和 `initialConfig` 属性，支持外部传入初始配置
- ConfigPanel 布局优化：日期一行、金额+费率一行、频率+定投日一行
- 组合回测各基金每月分配金额改为使用 ConfigPanel 中的基础每期金额
- 组合回测外层卡片嵌套修复
- 页面宽度改为 100%

### 技术
- 新增 `api_server.py` FastAPI 数据服务（搜索 + 拉取 + 聊天）
- 新增 `portfolio.ts` 组合回测引擎
- 新增 `pages/PortfolioBacktest.tsx` 组合回测页面
- 平台最大宽度从 1400px 改为 100%
- 配置面板策略区块改为复选框开关，默认关闭
- FundContribution 改为表格展示

# 组合回测页面 — 设计方案（草稿 v4）

## 概述
新增"组合回测"页面，布局与现有单基金回测保持一致。

## 页面布局（参照首页）

```
┌─ [单基金回测] ─ [组合回测] ─────────────────────────────┐
├──────────────┬──────────────────────────────────────────┤
│  基金列表      │  结果展示（回测后显示）                    │
│  ┌──────────┐ │  ┌────────┬────────┬────────┬────────┐  │
│  │ ＋ 添加   │ │  │总收益率 │年化收益 │总资产   │最大回撤│  │
│  ├──────────┤ │  ├────────┴────────┴────────┴────────┤  │
│  │ 📌 沪深300│ │  │  组合资产净值曲线                   │  │
│  │ 📌 纳指   │ │  │  📈                              │  │
│  │ 📌 黄金   │ │  └───────────────────────────────────┘  │
│  └──────────┘ │  ┌───────────────────────────────────┐  │
│               │  │  各基金收益贡献                     │  │
│  当前基金配置   │  │  沪深300 +¥12,000 (48%)           │  │
│  ┌──────────┐ │  │  纳指ETF   +¥8,000  (32%)          │  │
│  │ Config   │ │  │  黄金ETF   +¥5,000  (20%)          │  │
│  │ Panel    │ │  └───────────────────────────────────┘  │
│  │ (复用)   │ │                                         │
│  │          │ │                                         │
│  │ [▶ 开始] │ │                                         │
│  └──────────┘ │                                         │
└──────────────┴──────────────────────────────────────────┘
```

左侧固定宽度 340px，右侧自适应。完全复用首页的视觉风格。

## 可用资金配置

放在 Tab 栏下方、左右两栏上方，通栏显示：

```
┌─────────────────────────────────────────────────────────┐
│  初始金额 [¥100,000]   每月可用 [¥10,000]   已分配 ¥X/月   │
└─────────────────────────────────────────────────────────┘
```

## 交互流程

1. **顶部**：设置初始金额和每月可用金额
2. **左下**：先从预设/已添加基金中选取 → 设置每月分配金额 → 点击左侧列表中的基金 → 右侧 ConfigPanel 显示该基金的策略
3. **右上**：回测后展示组合指标 + 资产曲线 + 各基金收益贡献

## 数据流

```
可用资金 + 各基金分配方案 + 各基金策略参数
         ↓
组合引擎（逐月分配资金 → 各基金调用 runBacktest）
         ↓
各基金 BacktestResult → 按日汇总 → 组合 BacktestResult
         ↓
右上结果展示
```

# 项目概述

## 项目名称

ETF 基金回测工具 (ETF Fund Backtest Tool)

## 项目定位

一款面向个人投资者的 ETF/场外基金历史回测 Web 应用。用户可选择基金、配置投资策略，通过历史净值数据模拟收益表现，辅助投资决策。

## 核心功能

### 单基金回测
对单只基金进行历史数据回测，支持多种策略组合。

### 组合回测
构建多只基金的投资组合，配置每月资金分配，模拟组合整体收益。

### AI 投资顾问
接入 DeepSeek API，基于用户持仓和策略提供智能分析建议。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | React 18 + TypeScript |
| 构建工具 | Vite |
| UI 组件库 | Ant Design (antd) |
| 图表库 | ECharts |
| 后端框架 | FastAPI (Python) |
| 数据源 | AkShare (东方财富 / 天天基金网) |
| AI 模型 | DeepSeek Chat API |

## 数据源说明

通过 AkShare 库调用东方财富天天基金网接口，获取基金历史单位净值数据。数据字段包括：日期、单位净值、累计净值、日增长率。输出为类 K 线 CSV 格式（date/open/close/high/low/volume/amount），其中净值存储在 close 字段。
